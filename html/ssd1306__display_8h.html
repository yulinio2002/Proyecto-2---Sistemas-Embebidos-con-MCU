Detailed Description

Driver para display SSD1306 I2C para el sistema de control de acceso.

Author
Sistema de Control de Acceso
Date
2025

Driver simplificado del display SSD1306 OLED I2C para mostrar información del sistema de control de acceso incluyendo fecha/hora, mensajes de entrada y estado del sistema.
