Detailed Description

Controlador principal del sistema de control de acceso con FreeRTOS.

Author
Sistema de Control de Acceso
Date
2025

Este módulo implementa la máquina de estados principal del sistema de control de acceso utilizando FreeRTOS para comunicación entre tareas.
